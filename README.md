#   Block-Unblock


## To Block
```powershell

python block.py [website-link-no-https://]

```

#### Examples

```powershell
python block.py web.whatsapp.com
```
```powershell
python block.py www.youtube.com
```
## To Unblock

```powershell

python unblock.py [website-name]

```
#### Examples

```powershell
python unblock.py whatsapp
```
